# E-Shop Project

## Overview
E-Shop is an e-commerce web application developed using the **CodeIgniter** framework.  
The purpose of this project is to provide a complete online shopping platform where users can browse products, add items to the shopping cart, place orders, and interact with the store through a simple and organized interface.

The project also includes an **administration panel** that allows the admin to manage products, orders, categories, brands, pages, settings, users, languages, blog posts, and vendors.

In addition, the project contains:
- A customer-facing online store
- An admin dashboard
- A vendor management system
- A blog system
- A product API
- Custom templates and settings

---

## Project Idea
The main idea of this project is to build a **complete e-commerce system** that connects customers, administrators, and vendors in one platform.

From the customer side, the system allows users to:
- View products
- Read product details
- Add products to the shopping cart
- Complete the checkout process
- Contact the store
- Manage user accounts

From the admin side, the system allows administrators to:
- Add, edit, and remove products
- Manage orders and discounts
- Manage categories and brands
- Control store settings and appearance
- Manage users and pages
- Manage blog posts and vendors
- Handle language-related settings

This makes the project a practical implementation of a real-world online store using PHP and CodeIgniter.
